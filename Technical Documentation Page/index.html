Introduction

C++ is a general-purpose programming language created as an extension of the C language.

It supports procedural, object-oriented, and generic programming features.

C++ is widely used in system/software development, game engines, and competitive programming.

It provides low-level memory manipulation and high-level abstractions.

Developed by Bjarne Stroustrup, C++ was released in 1985.

#include <iostream> int main() { std::cout << "Hello, World!"; }
  • Fast and powerful
  • Supports OOP
  • Compiled language
  • Used in real-time systems
  • Memory management control
Variables and Data Types

Variables store data in memory that can be used and modified throughout a program.

Data types define the kind of value a variable can hold: integer, float, char, etc.

C++ has both fundamental and derived data types.

Variables must be declared before use with a type and an optional initializer.

Constants can be defined using the `const` keyword.

int age = 25; float score = 89.5;
  • int
  • float
  • char
  • bool
  • double
Control Structures

Control structures manage the flow of the program.

C++ supports conditional and loop-based control structures.

They help in decision-making and repetitive tasks.

`if`, `else`, `switch`, `while`, `for`, and `do-while` are key elements.

Nested control structures are allowed.

if (x > 0) { std::cout << "Positive"; } for (int i = 0; i < 5; i++) { std::cout << i; }
  • if-else
  • switch
  • while
  • do-while
  • for loop
Functions

Functions are reusable blocks of code that perform a specific task.

They help in modularizing the code and improving readability.

C++ supports function overloading and recursion.

Each function has a return type, name, and parameters.

The `main()` function is the entry point of a C++ program.

int add(int a, int b) { return a + b; } void greet() { std::cout << "Hello"; }
  • Return type
  • Function name
  • Parameters
  • Function body
  • Return statement
Object Oriented Programming

OOP is a paradigm that organizes code using objects and classes.

C++ supports all OOP principles: inheritance, polymorphism, encapsulation, and abstraction.

Classes are user-defined data types representing real-world entities.

Objects are instances of classes.

Access specifiers like public, private, and protected control visibility.

class Car { public: void drive() { std::cout << "Driving"; } }; Car c1; c1.drive();
  • Class
  • Object
  • Inheritance
  • Encapsulation
  • Polymorphism
